::: tip
使用饥荒管理平台App前，需要先部署饥荒管理平台
:::

#### 下载安装

前往[releases](https://github.com/miracleEverywhere/dst-management-platform-desktop/releases)页面下载安装包

![下载](assets/app-download.png)

::: tip
所有的安装包均为64位
:::

双击安装包按照提示安装即可

#### 获取饥荒管理平台令牌

登录饥荒管理平台，进入平台工具-平台令牌页面，创建一个令牌

![创建令牌](assets/app-token-create.png)

#### 新建DMP

运行饥荒管理平台App

![主界面](assets/app-entry.png)

::: tip
如果你是第一次使用App，仅有一个新建按钮
:::

点击 + 新建

IP就输入饥荒管理平台的公网IP，端口也是，默认为80

平台令牌就粘贴上一步创建的令牌

备注可不填

![新建](assets/app-create.png)

#### 进入

点击进入按钮，即可管理对应的饥荒管理平台，页面与网页版一致

![管理页面](assets/app-rooms.png)

#### App部分功能介绍

![按钮](assets/app-button.png)

#### 退出App

右键点击饥荒管理平台小图标，点击退出即可

![退出](assets/app-exit.png)

::: tip
点击窗口的 `×` 仅为关闭窗口，不会退出
:::
