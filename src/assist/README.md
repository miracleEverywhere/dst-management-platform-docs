## 搭建服务

:::tip
该服务为代搭建，你需要提供服务器
:::

帮你安装部署饥荒管理平台，你只需要像游戏中一样，配置房间，管理房间，进行游戏

本服务`10元`一次，搭建时间`10分钟`到`1小时`之间，搭建时长取决于你的服务器性能

## 服务流程

1. 发送邮件至`763483966@qq.com`，邮件内容如下：
```text
饥荒管理平台搭建服务
```

2. 作者回复后，回复以下内容：

> **云服IP**：你的云服IP
> **云服密码**：你的云服root密码
> **端口开通情况**：已开通/未开通

:::tip
为了提升搭建效率和游戏体验，请提前设置防火墙，开通tcp端口7777，udp端口11001,11002 (也可以让我帮你开通)
:::

3. 作者收到服务器信息邮件后，会向你发送付款方式

4. 付款完成后开始搭建，搭建结束后会将饥荒管理平台的`页面地址`、`平台用户`以及`平台密码`以邮件的形式回复给你，并帮你创建一个可以运行、可以加入的世界

## 注意事项

- 作者有时可能没法第一时间看到你的邮件，但在`12`小时内会进行回复

- 该服务为`代搭建`服务，需要你提供云服务器

- 云服务器的架构必须为`amd64(x86_64)`，系统推荐为`Ubuntu24`

- 需要提供`root`密码或者含有`su权限`的用户和密码，例如`ubuntu`用户

- 付款方式为`微信`支付或者`支付宝`支付

- 该服务仅为代搭建饥荒管理平台，`不包含`后续的模组冲突等问题处理
